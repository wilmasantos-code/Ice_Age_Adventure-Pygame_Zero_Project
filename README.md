#  Jogo da Era do Gelo (Ajude o Sid!)

Este é um jogo de aventura point-and-click feito em Python com a biblioteca **Pygame Zero**. O objetivo é ajudar o Sid a chegar aos seus amigos, desviando dos inimigos.

Este projeto foi criado para um teste de tutoria de Python, seguindo todos os requisitos, como:
* Uso exclusivo de PgZero, math e random
* Menu principal funcional
* Classes próprias para Player e Inimigos
* Animação de sprites (parado e andando)

## Como Rodar o Jogo

Para jogar, você precisa ter o Python e o `pgzero` instalados.

1.  **Instale o Pygame Zero:**
    ```bash
    pip install pgzero
    ```

2.  **Rode o jogo:**
    Execute o arquivo `main.py` usando o `pgzrun`.
    ```bash
    pgzrun main.py
    ```

##  Controles

* **Mouse:** Clique em qualquer lugar da tela para mover o Sid.
